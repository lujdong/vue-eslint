
配置 vue 项目中的个人 eslint 规则

### examples

```javascript
// bad
const a=1

// good
const a = 1;
```

```javascript
// bad
const fn=()=>{}

// good
const fn = () => { }

```

```javascript
// bad
const obj = {
  name:'John',
  age:30,
  city:'New York',
};

// good
const obj = {
  name: 'John',
  age: 30,
  city: 'New York',
};

// bad
const arr = [1,2,3,4,5];

// good
const arr = [1, 2, 3, 4, 5];
```
